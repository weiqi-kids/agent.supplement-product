## 1. 核心資訊

| 項目 | 內容 |
|------|------|
| **網站名稱** | 保健食品產品情報系統 |
| **一句話描述** | 跨國保健食品市場監測與成分趨勢分析平台 |
| **核心價值主張** | 整合五大市場（美、加、日、韓、台）官方資料庫，提供數據驅動的保健食品選購決策支援 |

---

## 2. 目標受眾

### 主要受眾 (Primary)

| 項目 | 描述 |
|------|------|
| **是誰** | 關注健康管理的消費者（30-55 歲，有保健食品購買意願） |
| **目標/需求** | 想了解特定保健成分（如魚油、葉黃素）的功效、選購要點、品牌比較 |
| **痛點** | 市面資訊零散、廣告與事實難辨、不知道如何選擇適合自己的產品 |
| **使用場景** | 購買保健食品前做功課、比較不同品牌、查詢成分功效與副作用 |

### 次要受眾 (Secondary)

| 項目 | 描述 |
|------|------|
| **是誰** | 保健產品產業從業人員（研發、行銷、採購） |
| **目標/需求** | 取得跨國市場情報、追蹤成分趨勢、競品分析 |
| **痛點** | 各國資料庫分散、語言障礙、缺乏整合分析 |
| **使用場景** | 市場研究、產品開發規劃、撰寫提案報告 |

---

## 3. 差異化定位

| 項目 | 內容 |
|------|------|
| **與競品的差異** | 1. 直接整合官方資料庫（非二手資訊）<br>2. 跨五國市場比較（非單一市場）<br>3. 數據驅動分析（非主觀評測） |
| **獨特優勢** | 自動化資料更新系統、42 萬+ 產品資料、14 種熱門成分追蹤 |
| **不做什麼** | 不做產品銷售、不做付費推薦、不做醫療診斷建議 |

## 7. 現況問題摘要

### 技術問題
- sitemap.xml 不存在（返回 404）
- 部分 guide 頁面 URL 路徑問題

### 內容問題
- 相關成分表有條目但無對應頁面連結
- 藥物交互表混入不相關成分文獻
- 部分市場數據極少（如日本外泌體僅 1 筆）

### 使用者體驗
- 決策樹依賴 JavaScript，無替代方案
- 劑型分類標準不一致（Softgel vs Capsule）
