# 日本 FNFC

**Foods with Function Claims（機能性表示食品）**

## 資料來源

- 官方網站：[消費者廳機能性表示食品](https://www.fld.caa.go.jp/caaks/cssc01/)
- 主管機關：日本消費者廳

## 資料特性

- 產品數量：1,500+
- 更新頻率：持續更新
- 資料內容：產品資訊、機能性成分、健康聲明、企業申報資料

## 涵蓋範圍

企業自行申報的機能性表示食品，特點：
- 企業自行提交科學證據
- 無需政府事前審查
- 可標示機能性聲明
- 申報制度較 FOSHU 靈活
