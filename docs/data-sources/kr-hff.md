# 韓國 HFF

**Health Functional Food Database（건강기능식품）**

## 資料來源

- 官方網站：[MFDS](https://www.foodsafetykorea.go.kr/)
- 主管機關：韓國食品醫藥品安全處（MFDS）

## 資料特性

- 產品數量：44,000+
- 更新頻率：持續更新
- 資料內容：產品資訊、機能性成分、健康聲明

## 涵蓋範圍

韓國核准的健康機能食品，包括：
- 維生素與礦物質
- 益生菌
- Omega-3 脂肪酸
- 紅蔘與人蔘製品
- 其他機能性成分
