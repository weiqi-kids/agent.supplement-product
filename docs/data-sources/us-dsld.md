# 美國 DSLD

**Dietary Supplement Label Database**

## 資料來源

- 官方網站：[NIH DSLD](https://dsld.od.nih.gov/)
- 主管機關：美國國家衛生研究院（NIH）

## 資料特性

- 產品數量：214,000+
- 更新頻率：持續更新
- 資料內容：產品標籤資訊、成分、劑量、製造商

## 涵蓋範圍

美國市場販售的膳食補充劑（Dietary Supplements），包括：
- 維生素與礦物質
- 草本與植物萃取
- 胺基酸與蛋白質
- 益生菌
- 其他膳食補充劑
