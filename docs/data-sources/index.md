# 資料來源

本系統整合五大市場的官方保健食品資料庫。

## 涵蓋範圍

| 國家 | 資料來源 | 說明 |
|------|----------|------|
| [美國](us-dsld) | DSLD | Dietary Supplement Label Database |
| [加拿大](ca-lnhpd) | LNHPD | Licensed Natural Health Products Database |
| [韓國](kr-hff) | HFF | Health Functional Food Database |
| [日本](jp-foshu) | FOSHU | Foods for Specified Health Uses |
| [日本](jp-fnfc) | FNFC | Foods with Function Claims |

## 資料特性

- **美國 DSLD**：產品數量最多，涵蓋膳食補充劑標籤詳細資訊
- **加拿大 LNHPD**：包含完整成分與劑量資訊
- **韓國 HFF**：包含機能性成分與健康聲明
- **日本 FOSHU**：經過嚴格審查的特定保健用食品
- **日本 FNFC**：企業自行申報的機能性表示食品
