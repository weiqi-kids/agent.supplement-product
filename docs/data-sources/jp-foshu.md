# 日本 FOSHU

**Foods for Specified Health Uses（特定保健用食品）**

## 資料來源

- 官方網站：[消費者廳](https://www.caa.go.jp/)
- 主管機關：日本消費者廳

## 資料特性

- 產品數量：1,000+
- 更新頻率：定期更新
- 資料內容：產品資訊、機能性成分、健康聲明、審查結果

## 涵蓋範圍

經日本政府嚴格審查核准的特定保健用食品，特點：
- 需提交人體試驗數據
- 需經過政府審查
- 可標示特定健康功效
- 審查標準嚴格，產品數量較少
