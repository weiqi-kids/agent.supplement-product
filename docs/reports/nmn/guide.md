# NMN 選購指南

本指南協助您根據需求選擇適合的 NMN 保健食品。

## 為什麼選擇 NMN？

### 與其他抗衰老方案比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **NMN** | NAD+ 直接前驅物、研究快速累積 | 價格較高、長期數據仍在收集 | 追求提升 NAD+ 水平者 |
| NR（菸鹼醯胺核苷） | 研究較早、價格稍低 | 機轉類似 NMN | 對 NMN 效果不佳者 |
| 菸鹼酸（維生素 B3） | 價格低廉、歷史悠久 | 轉換效率低、可能有潮紅反應 | 預算有限者 |
| 白藜蘆醇 | 價格實惠、抗氧化作用 | 吸收率低、非直接提升 NAD+ | 搭配 NMN 使用 |
| 外泌體 | 細胞修復作用 | 機轉不同、價格更高 | 追求細胞再生者 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 40 歲以上感受到體力下降者
- 追求延緩衰老的生物駭客族群
- 運動員或高強度工作者
- 對 NAD+ 科學有了解並認同者

**⚠️ 需諮詢醫師**：
- 正在服用藥物者
- 有慢性疾病者
- 癌症患者或癌症存活者

**❌ 不建議使用**：
- 孕婦、哺乳婦女
- 18 歲以下青少年
- 期望短期見效者（需持續補充）

## 常見問題 FAQ

### Q1: NMN 和 NR 哪個好？
**A:**
- **NMN**：分子較大，需經轉化後進入細胞，但近期研究顯示吸收效果良好
- **NR（菸鹼醯胺核苷）**：分子較小，較易進入細胞
- 建議：兩者效果相近，可依價格與個人反應選擇

### Q2: 什麼時候吃最好？
**A:** 建議早上空腹或餐前服用，可提升吸收率。避免晚上服用以免影響睡眠（部分人會感到精力提升）。

### Q3: NMN 需要搭配什麼成分？
**A:**
- **白藜蘆醇**：協同活化 SIRT1
- **輔酶 Q10**：支持粒線體能量產生
- **PQQ**：促進粒線體新生
- **TMG（甜菜鹼）**：補充甲基供體

### Q4: 有副作用嗎？
**A:** NMN 整體安全性良好，少數人可能出現：
- 輕微腸胃不適
- 初期精力提升影響睡眠
- 頭痛（通常因劑量過高）
建議從低劑量開始，逐步調整。

### Q5: 需要長期服用嗎？
**A:** NMN 不會在體內累積，停用後 NAD+ 水平會逐漸回到基線。若追求持續效果，建議長期補充。

### Q6: 如何判斷品質？
**A:**
- 查看第三方檢測報告（CoA）
- 確認 β-NMN 純度 ≥ 99%
- 選擇有 GMP 認證的品牌
- 注意保存條件（避光、低溫）
