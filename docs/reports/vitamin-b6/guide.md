# 維生素B6選購指南

本指南協助您根據需求選擇適合的維生素B6保健食品。

## 為什麼選擇維生素B6？

### 與其他B群維生素比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **單方B6** | 劑量精確、針對性補充 | 可能需搭配其他B群 | 明確B6需求者 |
| B群複方 | 協同作用、均衡補充 | 個別劑量較低 | 一般保養 |
| 天然食物 | 天然來源、無過量風險 | 攝取量難控制 | 飲食均衡者 |
| P-5-P（活性B6） | 無需肝臟轉換、直接利用 | 價格較高 | 肝功能不佳者 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 經前症候群（PMS）困擾者
- 孕婦（孕吐緩解）
- 老年人（吸收能力下降）
- 素食者
- 服用口服避孕藥者
- 蛋白質攝取量高者

**⚠️ 需諮詢醫師**：
- 服用帕金森氏症藥物者
- 服用抗癲癇藥物者
- 腎臟疾病患者

**❌ 不建議過量**：
- 長期高劑量（>100mg/日）可能導致神經病變

## 常見問題 FAQ

### Q1: 維生素B6和B群複方該選哪個？
**A:**
- **單方B6**：適合有明確B6需求者（如PMS、孕吐），可精確控制劑量
- **B群複方**：適合一般保養，B群各成分有協同作用
- 建議：若無特殊需求，選擇B群複方較均衡

### Q2: P-5-P 和一般 B6 有什麼差別？
**A:**
- **一般B6（鹽酸吡哆醇）**：需經肝臟轉換為活性型態才能利用
- **P-5-P（磷酸吡哆醛）**：已是活性型態，可直接被身體利用
- 適合選P-5-P：肝功能不佳者、追求更高生物利用度者

### Q3: 維生素B6對經前症候群有效嗎？
**A:** 研究顯示，每日50-100mg維生素B6可能緩解PMS症狀（如情緒波動、乳房脹痛）。建議在月經前一週開始補充，但應避免長期高劑量。

### Q4: 吃太多維生素B6會怎樣？
**A:** 長期服用高劑量（>200mg/日）可能導致周邊神經病變，症狀包括：
- 手腳麻木、刺痛
- 行走困難
- 感覺異常
停止補充後通常會逐漸恢復。每日補充應控制在100mg以下。

### Q5: 什麼時候吃B6最好？
**A:**
- **早上或隨餐**：B6可能影響睡眠，建議早上服用
- **避免睡前**：高劑量B6可能讓夢境更生動，影響睡眠品質
- **與其他B群一起**：B群最好一起服用，發揮協同作用
