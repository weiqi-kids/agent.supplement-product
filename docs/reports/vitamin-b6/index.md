# 維生素B6（Vitamin B6 / Pyridoxine）

維生素B6是水溶性B群維生素，參與超過 100 種酵素反應。本頁面追蹤全球維生素B6保健食品市場動態。

## 要解決什麼問題？

根據產品健康聲明分析，維生素B6產品主要訴求：

- **神經系統**：支持神經遞質合成（血清素、多巴胺）
- **蛋白質代謝**：參與胺基酸代謝
- **免疫功能**：維持免疫細胞正常運作
- **經前症候群**：緩解 PMS 症狀
- **孕期支持**：緩解孕吐（噁心嘔吐）

## 造成問題的原因

維生素B6攝取不足的原因：

- 老年人吸收能力下降
- 某些藥物增加B6需求（口服避孕藥、異煙肼）
- 素食者可能攝取不足（動物性來源生物利用度較高）
- 酒精干擾B6代謝
- 高蛋白飲食增加B6需求

## 作用機轉

維生素B6的生物活性機制：

1. **轉胺酶輔酶**：參與胺基酸代謝與轉換
2. **神經遞質合成**：血清素、多巴胺、GABA 合成所需
3. **血紅素合成**：參與紅血球生成
4. **糖原分解**：肝醣磷酸化酶的輔酶
5. **同半胱胺酸代謝**：與B12、葉酸協同降低同半胱胺酸

## 發展歷史與現況

**發展歷程**：
- 1934 年：發現維生素B6
- 1938 年：結構鑑定與命名
- 1960 年代：確認多種型態（吡哆醇、吡哆醛、吡哆胺）
- 1980 年代：P-5-P 活性型態研究
- 2000 年代：高劑量神經毒性警告建立
- 2020 年代：P-5-P 產品普及

**市場現況**（2026 年 2 月）：
- 美國市場占全球 65%
- 錠劑、膠囊為最主流劑型
- 多見於 B 群複方產品中
- P-5-P 活性型態受追捧
- PMS、孕吐用途單方產品穩定需求
