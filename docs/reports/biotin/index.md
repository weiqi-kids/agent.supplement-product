# 生物素

> 維生素 B7，頭髮、皮膚、指甲的美麗營養素

## 什麼是生物素？

生物素（Biotin）是維生素 B 群的一員，又稱維生素 B7 或維生素 H（H 源自德文 Haar und Haut，意為頭髮與皮膚）。它是一種水溶性維生素，在體內作為輔酶參與多種代謝反應。

生物素的主要功能：
- **能量代謝**：參與碳水化合物、脂肪、蛋白質代謝
- **角蛋白合成**：支持頭髮、皮膚、指甲健康
- **基因表達調控**：影響細胞功能
- **血糖調節**：輔助胰島素作用

## 為什麼需要補充？

### 可能需要補充的族群

| 族群 | 原因 |
|------|------|
| 頭髮稀疏/指甲脆弱者 | 角蛋白合成需求 |
| 孕婦/哺乳期 | 需求量增加，約 30% 孕婦有輕微缺乏 |
| 長期服用抗生素者 | 影響腸道菌群合成 |
| 長期生食蛋白者 | 生蛋白中的 Avidin 會結合生物素 |
| 酗酒者 | 吸收減少 |
| 特定遺傳疾病患者 | 生物素酶缺乏症 |

### 缺乏症狀

- 掉髮、頭髮稀疏
- 皮膚乾燥、皮疹（尤其眼、鼻、口周圍）
- 指甲脆弱易斷
- 疲勞、肌肉疼痛
- 情緒低落

## 市場概況

生物素是美容保健市場的熱門成分，常見於：

| 產品類型 | 說明 |
|----------|------|
| 髮膚甲複方 | Hair, Skin & Nails 配方 |
| 高劑量單方 | 5000-10000 mcg |
| B 群複方 | 含標準劑量生物素 |
| 美容軟糖 | 糖果劑型，方便食用 |

## 劑量範圍

| 類型 | 劑量 |
|------|------|
| 每日建議量（成人） | 30 mcg |
| 一般保健補充 | 30-100 mcg |
| 美容用途 | 2500-10000 mcg |

> 生物素為水溶性，過量會經尿液排出，目前無設定上限攝取量
