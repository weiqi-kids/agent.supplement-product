# 葡萄糖胺選購指南

本指南協助您根據個人需求選擇適合的葡萄糖胺產品。

## 選購決策樹

```
我想改善關節問題
    │
    ├─ 輕度不適（預防保養）
    │   └─ 單方葡萄糖胺 500-1000mg
    │
    ├─ 中度不適（明顯感受）
    │   └─ 葡萄糖胺 + 軟骨素複方
    │
    └─ 重度不適（影響生活）
        └─ 高劑量複方 + 薑黃素/MSM
            └─ 建議諮詢醫師
```

## 選購要點

### 1. 成分形式

| 形式 | 特點 | 適合對象 |
|------|------|----------|
| 葡萄糖胺硫酸鹽 | 研究最多、證據較充分 | 首選形式 |
| 葡萄糖胺鹽酸鹽 | 葡萄糖胺含量較高 | 對硫過敏者 |
| N-乙醯葡萄糖胺 | 較新形式 | 嘗試新配方者 |

### 2. 劑量建議

| 用途 | 每日劑量 | 說明 |
|------|----------|------|
| 預防保養 | 500-1000mg | 維持關節健康 |
| 輕度不適 | 1000-1500mg | 改善症狀 |
| 明顯症狀 | 1500mg | 臨床研究常用劑量 |

### 3. 複方搭配

| 搭配成分 | 功效 | 常見劑量 |
|----------|------|----------|
| 軟骨素 | 協同保護軟骨 | 800-1200mg |
| MSM | 抗發炎、減輕疼痛 | 1000-3000mg |
| 薑黃素 | 天然抗發炎 | 200-500mg |
| 玻尿酸 | 關節潤滑 | 60-200mg |

### 4. 來源選擇

| 來源 | 優點 | 注意事項 |
|------|------|----------|

| 植物發酵 | 素食可用、無過敏原 | 價格較高 |

## 常見問題 FAQ

### Q1: 葡萄糖胺要吃多久才有效？
建議持續補充 4-8 週以上才能評估效果。關節組織更新需要時間，短期使用可能看不到明顯改善。

### Q2: 葡萄糖胺有副作用嗎？
大部分人耐受性良好。少數人可能有輕微腸胃不適、脹氣等症狀，建議隨餐服用以減少不適。

### Q3: 蝦蟹過敏可以吃葡萄糖胺嗎？
傳統葡萄糖胺萃取自蝦蟹殼，過敏風險較低（過敏原主要在肉質）。但建議選擇植物發酵來源更安心。

### Q4: 葡萄糖胺和軟骨素要一起吃嗎？
研究顯示兩者併用效果較佳。若預算許可，建議選擇含軟骨素的複方產品。

### Q5: 糖尿病患者可以吃葡萄糖胺嗎？
一般劑量對血糖影響不大，但建議糖尿病患者使用前諮詢醫師，並定期監測血糖。

### Q6: 痛寶精和 Move Free 哪個好？
兩者都是市場知名品牌，成分配方略有不同。選擇時可比較：
- 葡萄糖胺含量與形式
- 複方成分（軟骨素、MSM等）
- 價格與每日攝取量

## 警語與注意事項

- 本指南僅供參考，不構成醫療建議
- 嚴重關節問題應就醫診斷
- 服用抗凝血藥物者請先諮詢醫師
- 孕婦、哺乳婦女使用前請諮詢醫師
- 甲殼類過敏者請選擇植物來源產品
