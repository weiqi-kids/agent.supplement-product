# 菸鹼酸

> 維生素 B3，能量代謝與心血管健康的重要營養素

## 什麼是菸鹼酸？

菸鹼酸（Niacin）是維生素 B 群的一員，又稱維生素 B3 或菸酸。它有兩種主要形式：菸鹼酸（Nicotinic Acid）和菸鹼醯胺（Nicotinamide/Niacinamide）。菸鹼酸是合成 NAD（菸鹼醯胺腺嘌呤二核苷酸）的前驅物，參與體內超過 400 種酵素反應。

菸鹼酸的主要功能：
- **能量代謝**：參與碳水化合物、脂肪、蛋白質的能量轉換
- **DNA 修復**：NAD+ 參與 DNA 損傷修復機制
- **血脂調節**：高劑量菸鹼酸可調節膽固醇
- **皮膚健康**：維持皮膚屏障功能
- **神經系統**：支持神經系統正常運作

## 為什麼需要補充？

### 可能需要補充的族群

| 族群 | 原因 |
|------|------|
| 飲食不均衡者 | 攝取不足 |
| 酗酒者 | 吸收減少，代謝增加 |
| 血脂異常者 | 高劑量菸鹼酸可調節血脂（需醫師處方） |
| 糙皮病患者 | 嚴重缺乏導致的疾病 |
| 長期服用特定藥物者 | 某些藥物影響菸鹼酸代謝 |

### 缺乏症狀

- **糙皮病（Pellagra）**：嚴重缺乏的典型症狀
  - 皮膚炎（Dermatitis）
  - 腹瀉（Diarrhea）
  - 失智（Dementia）
- 疲勞、虛弱
- 消化不良
- 口腔潰瘍

## 菸鹼酸 vs 菸鹼醯胺

| 特性 | 菸鹼酸（Nicotinic Acid） | 菸鹼醯胺（Nicotinamide） |
|------|--------------------------|--------------------------|
| 降血脂作用 | ✅ 有（高劑量） | ❌ 無 |
| 潮紅反應 | ⚠️ 常見 | ❌ 無 |
| 皮膚保養 | 一般 | ✅ 常用於護膚品 |
| NAD+ 轉換 | ✅ 可 | ✅ 可 |

## 市場概況

菸鹼酸產品主要分為：

| 產品類型 | 說明 |
|----------|------|
| 一般劑量 B3 | 每日建議量，一般保健 |
| 高劑量菸鹼酸 | 血脂調節用途（需處方） |
| 緩釋型 | 減少潮紅反應 |
| 無潮紅配方 | Flush-Free，使用菸鹼醯胺或肌醇六菸鹼酸酯 |
| B 群複方 | 含標準劑量菸鹼酸 |

## 劑量範圍

| 類型 | 劑量 |
|------|------|
| 每日建議量（成人） | 14-16 mg |
| 一般保健 | 20-50 mg |
| 血脂調節（處方） | 1000-3000 mg |
| 上限攝取量（UL） | 35 mg（一般補充形式） |

> ⚠️ 高劑量菸鹼酸（>100 mg）可能引起潮紅反應，應在醫師指導下使用
