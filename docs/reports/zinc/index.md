# 鋅（Zinc）

鋅是人體必需的微量礦物質，參與超過 300 種酵素反應。本頁面追蹤全球鋅保健食品市場動態。

## 要解決什麼問題？

根據產品健康聲明分析，鋅產品主要訴求：

- **免疫功能**：支持免疫細胞發育與功能
- **傷口癒合**：參與皮膚修復與膠原蛋白合成
- **生殖健康**：維持正常睾固酮濃度
- **味覺嗅覺**：維持感官功能正常
- **皮膚健康**：調節皮脂分泌、改善痘痘

## 造成問題的原因

鋅攝取不足的原因：

- 素食飲食（植物性來源吸收率較低）
- 植酸、草酸等抗營養因子抑制吸收
- 老年人吸收能力下降
- 腸胃道疾病影響吸收
- 高度運動者流汗流失增加

## 作用機轉

鋅的生物活性機制：

1. **酵素輔因子**：參與 300+ 種酵素反應
2. **DNA/RNA 合成**：細胞分裂必需
3. **蛋白質合成**：支持組織修復與生長
4. **免疫調節**：T 細胞發育與功能所需
5. **抗氧化**：SOD（超氧化物歧化酶）組成成分

## 發展歷史與現況

**發展歷程**：
- 1869 年：發現鋅對生物體的重要性
- 1961 年：確認鋅缺乏會導致生長遲緩
- 1980 年代：鋅喉糖用於感冒研究開始
- 1990 年代：螯合型鋅提高生物利用度
- 2000 年代：鋅與免疫功能研究深入
- 2020 年代：COVID-19 期間鋅補充劑需求激增

**市場現況**（2026 年 2 月）：
- 美國、韓國為主要市場
- 錠劑、膠囊為最主流劑型
- Nature's Way、NOW 為領導品牌
- 鋅 + 維生素C 免疫複方持續熱銷
- 喉糖型態感冒季節需求高
