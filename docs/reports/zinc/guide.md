# 鋅選購指南

本指南協助您根據需求選擇適合的鋅保健食品。

## 為什麼選擇鋅？

### 與其他免疫營養素比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **鋅補充劑** | 劑量精確、研究充分 | 過量可能抑制銅吸收 | 免疫支持、傷口癒合 |
| 天然食物（牡蠣、肉類） | 天然來源、含其他營養 | 素食者攝取不足 | 飲食均衡者 |
| 複方免疫配方 | 多元營養素組合 | 個別劑量較低 | 追求綜合保養 |
| 維生素C | 抗氧化、協同作用 | 作用機轉不同 | 搭配使用更佳 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 素食者（植物性飲食鋅吸收較差）
- 老年人（吸收能力下降）
- 孕婦、哺乳期婦女
- 頻繁感冒者
- 皮膚問題（痘痘、傷口癒合慢）

**⚠️ 需諮詢醫師**：
- 服用抗生素者
- 長期補充銅者
- 腎臟疾病患者

**❌ 不建議過量**：
- 長期高劑量可能導致銅缺乏

| 族群 | 每日建議量 |
|------|-----------|
| 成年男性 | 11 mg |

### 4. 品質指標

- **型態標示**：避免氧化鋅，優先選螯合型或葡萄糖酸鋅
- **銅搭配**：高劑量產品應含銅
- **認證**：USP、NSF、GMP 認證
- **無添加**：避免過多填充劑

## 常見問題 FAQ

### Q1: 感冒吃鋅有效嗎？
**A:** 研究顯示，感冒初期24小時內開始補充鋅喉糖（葡萄糖酸鋅），可縮短感冒病程約1-2天。但需在症狀出現後立即使用，延遲服用效果有限。

### Q2: 鋅可以長期吃嗎？
**A:** 適量補充（15mg以下）可長期服用。但若長期補充高劑量（>25mg/日），應同時補充銅（1-2mg）以避免銅缺乏。銅缺乏可能導致貧血和神經問題。

### Q3: 空腹吃鋅會不舒服嗎？
**A:** 部分人空腹服用鋅會噁心。建議：
- 隨餐服用或餐後服用
- 選擇溫和型態（如葡萄糖酸鋅、螯合鋅）
- 避免一次高劑量

### Q4: 鋅和維生素C可以一起吃嗎？
**A:** 可以，且常見於免疫複方產品。兩者在免疫功能上有協同作用，一起服用不會互相干擾。

### Q5: 素食者需要額外補鋅嗎？
**A:** 建議考慮補充。植物性飲食中的植酸會降低鋅吸收，素食者鋅攝取建議量約為一般人的1.5倍。可選擇螯合型鋅提高吸收率。
