# 維生素C選購指南

本指南協助您根據需求選擇適合的維生素C保健食品。

## 為什麼選擇維生素C？

### 與其他抗氧化劑來源比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **維生素C補充劑** | 劑量明確、吸收穩定、研究充分 | 高劑量可能引起腸胃不適 | 日常保養、免疫力提升 |
| 天然食物 | 含有其他營養素、無過量風險 | 攝取量難以精確控制 | 飲食均衡者 |
| 脂溶性維生素C | 溫和不刺激、停留時間長 | 價格較高 | 腸胃敏感者 |
| 複方配方 | 搭配鋅、生物類黃酮效果更佳 | 成分複雜 | 追求綜合效果者 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 抽菸者（菸草會消耗維生素C）
- 壓力大、生活不規律者
- 蔬果攝取不足者
- 術後恢復期
- 追求皮膚保養者

**⚠️ 需諮詢醫師**：
- 腎結石病史者（高劑量可能增加結石風險）
- 血色素沉著症患者
- 服用抗凝血藥物者

**❌ 不建議使用**：
- G6PD缺乏症患者（高劑量靜脈注射）

## 常見問題 FAQ

### Q1: 維生素C一天吃多少才夠？
**A:** 一般成人每日建議量約 100mg，但用於免疫保養可提高至 500-1000mg。上限量為 2000mg/日，超過可能引起腸胃不適或腹瀉。

### Q2: 什麼時候吃維生素C最好？
**A:** 建議隨餐服用，可減少腸胃刺激。若服用高劑量，可分次服用（如早晚各一次）以提高吸收效率。

### Q3: 天然維生素C和合成的有差別嗎？
**A:** 化學結構相同，人體無法區分。但天然來源產品通常含有生物類黃酮等協同成分，可能有助於吸收。

### Q4: 維生素C會不會吃太多？
**A:** 維生素C是水溶性，過量會從尿液排出。但長期高劑量（>2000mg/日）可能增加腎結石風險，腸胃敏感者可能出現腹瀉。

### Q5: 感冒時吃維生素C有效嗎？
**A:** 研究顯示，日常補充維生素C可能縮短感冒病程約8%，但感冒後才補充效果有限。高壓力族群（如運動員）受益較明顯。

### 安全建議

1. **諮詢醫師**：服用處方藥物者，補充保健食品前應諮詢醫師
2. **注意劑量**：遵循建議劑量，避免長期高劑量補充
3. **觀察反應**：開始補充時注意身體反應，如有腹瀉或不適應調整劑量
4. **腎結石風險**：有結石病史者應控制劑量在1000mg/日以下
