# 維生素C（Vitamin C / Ascorbic Acid）

維生素C是人體必需的水溶性維生素，具有強效抗氧化作用。本頁面追蹤全球維生素C保健食品市場動態。

## 要解決什麼問題？

根據產品健康聲明分析，維生素C產品主要訴求：

- **免疫支持**：增強免疫細胞功能、縮短感冒病程
- **抗氧化**：清除自由基、保護細胞
- **膠原蛋白合成**：維持皮膚彈性、傷口癒合
- **鐵質吸收**：提高非血紅素鐵吸收率
- **牙齦健康**：維持結締組織完整性

## 造成問題的原因

維生素C攝取不足的原因：

- 蔬果攝取量不足
- 抽菸者維生素C消耗增加（每天額外流失 25-35mg）
- 壓力大、生活不規律
- 烹調方式破壞維生素C（加熱、長時間浸泡）
- 某些疾病狀態（如糖尿病）需求量增加

## 作用機轉

維生素C的生物活性機制：

1. **電子供體**：作為抗氧化劑，中和自由基
2. **輔酶功能**：參與膠原蛋白、肉鹼、神經遞質合成
3. **金屬離子還原**：將三價鐵還原為二價鐵，促進吸收
4. **免疫調節**：增強白血球吞噬能力、促進干擾素產生
5. **再生維生素E**：將氧化型維生素E還原為活性型

## 發展歷史與現況

**發展歷程**：
- 1747 年：發現柑橘可預防壞血病
- 1932 年：Albert Szent-Györgyi 分離出維生素C
- 1970 年代：Linus Pauling 推廣高劑量維生素C
- 1990 年代：酯化C 等新型態問世
- 2000 年代：與生物類黃酮複方普及
- 2020 年代：緩釋型、脂質體維生素C 興起

**市場現況**（2026 年 2 月）：
- 美國市場占全球 67%
- 錠劑、膠囊為最主流劑型
- Nature's Way、Douglas Laboratories 為領導品牌
- 高劑量（1000mg+）產品持續熱銷
- 維生素C + 鋅 免疫複方受歡迎
