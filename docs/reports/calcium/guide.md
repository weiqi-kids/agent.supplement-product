# 鈣選購指南

本指南協助您根據需求選擇適合的鈣質保健食品。

## 為什麼選擇鈣？

### 與其他鈣來源比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **鈣補充劑** | 劑量精確、方便補充 | 需注意與其他礦物質平衡 | 飲食鈣攝取不足者 |
| 乳製品 | 天然來源、含維生素D | 乳糖不耐者不適用 | 可接受乳製品者 |
| 高鈣蔬菜 | 無乳糖問題、含其他營養 | 吸收率較低、需大量攝取 | 素食者 |
| 強化食品 | 自然融入飲食 | 難精確控制劑量 | 日常保養 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 更年期婦女（骨質流失加速）
- 乳糖不耐者
- 素食者
- 青少年（骨骼發育期）
- 老年人

**⚠️ 需諮詢醫師**：
- 腎臟疾病患者
- 副甲狀腺功能異常者
- 服用心臟藥物者
- 有腎結石病史者

**❌ 不建議過量**：
- 高血鈣症患者

## 常見問題 FAQ

### Q1: 碳酸鈣和檸檬酸鈣該選哪個？
**A:**
- **碳酸鈣**：元素鈣含量高(40%)、價格實惠，但需隨餐服用（需要胃酸幫助吸收）
- **檸檬酸鈣**：不需胃酸、可空腹服用、較不易脹氣，適合老年人或服用制酸劑者
- 建議：胃酸正常選碳酸鈣，胃酸不足或常空腹服用選檸檬酸鈣

### Q2: 鈣和鐵可以一起吃嗎？
**A:** 不建議同時服用。鈣會降低鐵的吸收，建議錯開至少2小時。例如早餐吃鐵、晚餐後吃鈣。

### Q3: 一天中什麼時候吃鈣最好？
**A:**
- **碳酸鈣**：隨餐服用（需胃酸）
- **檸檬酸鈣**：任何時間皆可
- 若劑量超過500mg，建議分次服用（如早、晚各一次）

### Q4: 吃鈣會造成結石嗎？
**A:** 研究顯示，適量補鈣（<2500mg/日）不會增加結石風險。反而飲食鈣攝取不足可能增加結石機率。但需注意補充足夠水分，避免過量維生素D。

### Q5: 素食者適合哪種鈣？
**A:** 建議選擇海藻鈣或檸檬酸鈣，避免動物來源（如牡蠣鈣）。海藻鈣為植物性來源，且含有多種微量礦物質。

## ⚠️ 藥物交互提醒

本章節整理鈣相關的藥物交互資訊，供參考。

### 已知交互作用

| 藥物類型 | 交互影響 | 建議 |
|----------|----------|------|
| **雙磷酸鹽類（骨鬆藥）** | 鈣降低藥物吸收 | 間隔至少2小時 |
| **四環黴素類抗生素** | 鈣降低抗生素吸收 | 間隔至少2-4小時 |
| **喹諾酮類抗生素** | 鈣降低抗生素吸收 | 間隔至少2小時 |
| **甲狀腺藥物（Levothyroxine）** | 鈣降低藥物吸收 | 間隔至少4小時 |
| **鐵劑** | 相互降低吸收 | 間隔至少2小時 |
| **利尿劑（Thiazide）** | 可能增加血鈣 | 監測血鈣值 |
| **毛地黃類心臟藥** | 高血鈣增加毒性風險 | 諮詢醫師 |

### 安全建議

1. **諮詢醫師**：服用處方藥物者，補充鈣劑前應諮詢醫師
2. **錯開服藥**：與大多數藥物間隔2小時以上服用
3. **分次服用**：每次不超過500mg，提高吸收效率
4. **搭配維生素D**：確保維生素D充足以利鈣吸收
5. **適量攝取**：勿超過每日上限量2500mg

> ⚠️ **免責聲明**：本資訊僅供教育和研究目的，不構成醫療建議。任何用藥或補充劑變更應諮詢專業醫療人員。
