# 鈣（Calcium）

鈣是人體含量最豐富的礦物質，約 99% 存在於骨骼和牙齒中。本頁面追蹤全球鈣質保健食品市場動態。

## 要解決什麼問題？

根據產品健康聲明分析，鈣產品主要訴求：

- **骨骼健康**：維持骨密度、預防骨質疏鬆
- **牙齒保健**：維持牙齒堅固
- **肌肉功能**：支持肌肉正常收縮
- **神經傳導**：維持神經系統正常運作
- **心臟功能**：參與心肌收縮調節

## 造成問題的原因

現代人鈣質攝取不足的原因：

- 乳製品攝取減少（乳糖不耐、素食）
- 高鈉飲食增加鈣流失
- 咖啡因與碳酸飲料影響鈣吸收
- 缺乏日照導致維生素D不足，影響鈣吸收
- 更年期婦女雌激素下降，骨質流失加速

## 作用機轉

鈣的生物活性機制：

1. **骨骼礦化**：與磷結合形成羥磷灰石，構成骨骼主要成分
2. **肌肉收縮**：鈣離子觸發肌動蛋白與肌凝蛋白結合
3. **神經傳導**：調節神經遞質釋放
4. **血液凝固**：參與凝血因子活化
5. **酵素活化**：作為多種酵素的輔因子

## 發展歷史與現況

**發展歷程**：
- 1930 年代：發現鈣與維生素D對骨骼健康的重要性
- 1960 年代：碳酸鈣補充劑普及
- 1990 年代：檸檬酸鈣等高吸收型態問世
- 2000 年代：鈣+D3 複方成為主流
- 2010 年代：維生素K2 的協同作用受關注
- 2020 年代：海藻鈣等天然來源興起

**市場現況**（2026 年 2 月）：
- 美國市場占全球 66%
- 錠劑為最主流劑型（34%）
- NOW、Douglas Laboratories 為領導品牌
- 鈣+D3+K2 複方產品持續成長
- 植物來源（海藻鈣）受素食者歡迎
