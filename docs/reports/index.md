# 報告總覽

本系統提供兩種定期報告，協助您掌握保健食品市場動態。

## 報告類型

| 報告 | 內容 | 更新頻率 |
|------|------|----------|
| [市場快照](market-snapshot/) | 各國市場產品統計、品類分佈、市場亮點 | 每週 |
| [成分雷達](ingredient-radar/) | 跨國成分趨勢、新興成分、成分熱度排行 | 每月 |

## 閱讀指南

- **市場快照**：適合快速了解各國市場最新動態與品類變化
- **成分雷達**：適合深入分析成分趨勢與跨國比較
