## 趨勢觀察

### 跨國共同趨勢

基礎維生素礦物質組合（Vitamin C + Calcium + Vitamin B6/B12 + Zinc + Magnesium）在美、加、韓三大市場持續穩居主流地位。本月最顯著的變化是 Vitamin D 在合併 Vitamin D3 和 Cholecalciferol 後，全球排名從第 8 位躍升至第 2 位（50,237 筆），超越 Calcium（46,861 筆），這反映了 Vitamin D 在各種複合配方中的廣泛添加以及三大市場消費者對免疫與骨骼健康的持續關注。Probiotics 相關成分在美國（約 5,915 件含益生菌產品）和韓國（1,746 件）持續強勢，腸道健康議題已成全球保健食品的核心訴求之一。

### 市場獨特趨勢

美國市場本月 DSLD 資料庫新增約 9,000 筆產品（205,738 筆升至 214,780 筆），增幅約 4.4%，新增產品廣泛分布於 specialty（76,497 件）和 botanicals（67,449 件）兩大品類，反映美國膳食補充劑市場的持續擴張。韓國市場的一個特殊現象是機能性成分「鐵（철）」以 36,489 件高居韓國資料登記數量之首，遠超 Zinc（5,790 件），這是因為韓國 HFF 資料中許多產品將鐵作為必要機能性成分標註。日本市場 Indigestible Dextrin 合計達 461 件（FOSHU 349 件 + FNFC 112 件含變體），應用範圍橫跨腸道健康、餐後血糖控制、血中中性脂肪控制等多項機能性宣稱。加拿大市場結構穩定，Vitamin C 與 Vitamin D 差距僅 35 件（16,964 vs 16,929），兩者在加拿大市場的競爭格局持續膠著。

### 值得關注的成分

以下成分值得深入追蹤：

1. **Vitamin D** — 高成長成分
   - 涵蓋市場：US（28,410 件）、CA（16,929 件）、KR（4,898 件）
   - 所屬品類：vitamins_minerals
   - 關注原因：本月全球排名從第 8 位躍升至第 2 位，主因為合併計算 Vitamin D3/Cholecalciferol 後的真實市場規模首次完整呈現。Vitamin D 在三大市場均位列前 6，且加拿大以其作為事實上的第一大成分（與 Vitamin C 幾乎並列），反映高緯度國家對 Vitamin D 補充的公衛政策效果
   - 追蹤建議：監測各國 Vitamin D 新品登記趨勢，評估 Vitamin D 與免疫健康宣稱的關聯性變化

2. **GABA** — 跨國潛力成分
   - 涵蓋市場：JP（FNFC 112 件）
   - 所屬品類：specialty
   - 關注原因：日本 FNFC 中最熱門的非纖維機能性成分，維持穩定市場規模。睡眠改善與壓力舒緩訴求在全球消費者中持續升溫，但歐美市場 GABA 仍以膳食補充劑形式銷售，官方認可規模遠小於日本
   - 追蹤建議：監測韓國 HFF 是否出現 GABA 機能性成分登記，評估加拿大 NPN 中 GABA 產品的增長趨勢

3. **Indigestible Dextrin** — 日本市場主導成分
   - 涵蓋市場：JP（FOSHU + FNFC 461 件）
   - 所屬品類：botanicals
   - 關注原因：日本市場產品數最多的單一成分，應用範圍橫跨腸道健康、餐後血糖控制、血中中性脂肪控制等多項機能性宣稱，且在 FOSHU 和 FNFC 雙軌制度下均獲大量認可
   - 追蹤建議：關注台灣衛福部及韓國 MFDS 是否有類似成分的機能性宣稱認可動態

4. **Monoglucosyl Hesperidin** — 新興柑橘類機能成分
   - 涵蓋市場：JP（FOSHU 12 件 + FNFC 21 件 = 33 件）
   - 所屬品類：specialty
   - 關注原因：糖轉移橙皮苷（モノグルコシルヘスペリジン）在日本 FOSHU 與 FNFC 雙系統中同時獲認可，主要訴求為改善末梢血流和降低中性脂肪，是少數同時跨越兩個日本機能性食品制度的成分，顯示其臨床證據基礎相對紮實
   - 追蹤建議：觀察此成分在韓國及東南亞市場的法規認可進展，評估其作為心血管健康成分的跨國推廣潛力

## 方法論說明

- **成分名稱標準化方法**：採用同義詞合併策略，將不同語言及學名俗名對應至統一標準名稱（如 Vitamin D3 = Cholecalciferol = コレカルシフェロール 統一計入 Vitamin D；Folic Acid 與 Folate 合併；Niacinamide 與 Niacin 合併；韓文 아연 對照至 Zinc）
- **日文成分名對照**：本月對照項目包含 GABA（γ-アミノ酪酸）、ルテイン（Lutein）、ヒアルロン酸Na（Hyaluronic Acid）、L-テアニン（L-Theanine）、難消化性デキストリン（Indigestible Dextrin）、葛イソフラボン（Kudzu Isoflavone）、茶カテキン（Tea Catechins）、モノグルコシルヘスペリジン（Monoglucosyl Hesperidin）等主要 FNFC/FOSHU 機能性成分
- **韓文成分名對照**：아연（Zinc）、비타민D（Vitamin D）、프로바이오틱스（Probiotics）、엽산（Folate）、철（Iron）、비타민B1（Vitamin B1）等韓文名稱對照英文標準名
- **已知限制**：
  - 美國 DSLD 包含完整的產品成分清單（含營養標示項目），本報告已過濾 Calories、Total Fat、Sodium、Total Carbohydrates 等純營養標示項目，僅保留具機能性的活性成分
  - 韓國 HFF 以 [] 標注機能性成分，統計邏輯與美加不同（美加為所有成分，韓國為申請認定的機能性成分），跨國絕對數量比較應考慮此差異
  - 韓國「鐵（철）」的高登記數量反映韓國 HFF 資料結構特性（大量產品將鐵標為機能性成分），不宜與美加的 Iron 數量直接比較
  - 加拿大 LNHPD 部分產品成分資訊以劑量格式列於 API（如 "Fish oil: 200.0 mg"），統計時以成分名稱歸併
  - 日本市場（FOSHU + FNFC 合計 2,603 件）資料量遠小於美加韓，跨國排名比較應考慮基數差異
