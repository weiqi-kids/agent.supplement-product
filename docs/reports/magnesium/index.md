# 鎂（Magnesium）

鎂是人體第四豐富的礦物質，參與超過 600 種生化反應。本頁面追蹤全球鎂保健食品市場動態。

## 要解決什麼問題？

根據產品健康聲明分析，鎂產品主要訴求：

- **肌肉放鬆**：緩解肌肉痙攣、抽筋
- **神經系統**：支持神經傳導、緩解焦慮
- **睡眠品質**：幫助入睡、改善睡眠深度
- **能量代謝**：參與 ATP 生成
- **心血管健康**：維持正常心律、血壓調節

## 造成問題的原因

鎂攝取不足的原因：

- 精緻飲食導致鎂攝取降低（穀物加工損失 80% 鎂）
- 土壤礦物質枯竭，作物鎂含量下降
- 壓力增加鎂消耗
- 某些藥物（利尿劑、PPI）增加鎂流失
- 高糖飲食增加鎂排泄

## 作用機轉

鎂的生物活性機制：

1. **ATP 輔因子**：所有 ATP 依賴反應都需要鎂
2. **肌肉放鬆**：對抗鈣的肌肉收縮作用
3. **神經調節**：調控 NMDA 受體，減少神經興奮
4. **血管舒張**：維持血管彈性與正常血壓
5. **DNA 穩定**：參與 DNA 修復與蛋白質合成

## 發展歷史與現況

**發展歷程**：
- 1808 年：Humphry Davy 分離出鎂元素
- 1950 年代：確認鎂對心臟功能的重要性
- 1980 年代：鎂缺乏與慢性病關聯研究
- 2000 年代：多種螯合型態問世
- 2010 年代：蘇糖酸鎂（Magtein）研發
- 2020 年代：助眠、抗焦慮功能受關注

**市場現況**（2026 年 2 月）：
- 美國市場占全球 71%
- 錠劑、膠囊、粉末均為主流劑型
- Douglas Laboratories、NOW 為領導品牌
- 甘胺酸鎂（助眠）需求快速成長
- 蘇糖酸鎂（認知）新興市場
