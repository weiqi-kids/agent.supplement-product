# 鎂選購指南

本指南協助您根據需求選擇適合的鎂保健食品。

## 為什麼選擇鎂？

### 與其他鎂來源比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **鎂補充劑** | 劑量精確、型態多元 | 部分型態可能腹瀉 | 明確缺乏者、特定需求 |
| 天然食物（堅果、深綠蔬菜） | 天然來源、含其他營養 | 現代飲食攝取常不足 | 飲食均衡者 |
| 瀉鹽浴（硫酸鎂） | 外用、放鬆肌肉 | 吸收量有限 | 局部放鬆需求 |
| 鎂油噴霧 | 經皮吸收 | 研究有限、可能刺激 | 不耐口服者 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 肌肉痙攣、抽筋頻繁者
- 睡眠品質不佳者
- 壓力大、焦慮者
- 偏頭痛患者
- 糖尿病患者（常見缺乏）
- 經期不適者

**⚠️ 需諮詢醫師**：
- 腎臟疾病患者
- 心律不整者
- 服用心臟藥物者
- 服用抗生素者

**❌ 不建議過量**：
- 腎功能不全者（排泄能力下降）

## 常見問題 FAQ

### Q1: 鎂的型態這麼多，該怎麼選？
**A:** 依主要需求選擇：
- **助眠**：甘胺酸鎂（最溫和、助眠效果最佳）
- **一般補充**：檸檬酸鎂（吸收好、價格適中）
- **便秘改善**：氧化鎂或檸檬酸鎂
- **運動/肌肉**：蘋果酸鎂
- **認知/記憶**：蘇糖酸鎂

### Q2: 吃鎂會拉肚子嗎？
**A:** 部分型態（如氧化鎂、檸檬酸鎂）具有輕微瀉下作用。建議：
- 從低劑量開始（100mg）
- 選擇甘胺酸鎂等溫和型態
- 分次服用
- 隨餐服用

### Q3: 鎂和鈣要一起吃還是分開？
**A:** 兩種說法都有，但研究顯示一起吃不會明顯影響吸收。若服用高劑量（各>500mg），可分開服用。理想的鈣鎂比例約 2:1。

### Q4: 什麼時候吃鎂最好？
**A:**
- **助眠目的**：睡前1小時服用甘胺酸鎂
- **一般補充**：隨餐服用減少腸胃不適
- **便秘用途**：空腹服用（通便效果更明顯）
- **運動支持**：運動前後

### Q5: 如何知道自己缺鎂？
**A:** 常見缺乏症狀包括：
- 肌肉痙攣、抽筋
- 眼皮跳動
- 疲勞、虛弱
- 睡眠困難
- 焦慮、易怒
- 心悸
建議檢測血中鎂離子濃度（但血鎂可能無法完全反映體內鎂狀態）。

## ⚠️ 藥物交互提醒

本章節整理鎂相關的藥物交互資訊，供參考。

### 已知交互作用

| 藥物類型 | 交互影響 | 建議 |
|----------|----------|------|
| **雙磷酸鹽類（骨鬆藥）** | 鎂降低藥物吸收 | 間隔至少2小時 |
| **四環黴素類抗生素** | 鎂降低抗生素吸收 | 間隔至少2-4小時 |
| **喹諾酮類抗生素** | 鎂降低抗生素吸收 | 間隔至少2小時 |
| **利尿劑** | 增加鎂流失 | 可能需額外補充 |
| **氫離子幫浦抑制劑（PPI）** | 長期使用降低鎂吸收 | 監測鎂濃度 |
| **肌肉鬆弛劑** | 鎂可能增強藥效 | 諮詢醫師 |
| **降血壓藥（鈣離子阻斷劑）** | 鎂可能增強降壓效果 | 監測血壓 |

### 安全建議

1. **諮詢醫師**：服用處方藥物者，補充鎂前應諮詢醫師
2. **錯開服藥**：與抗生素、骨鬆藥間隔至少2小時
3. **腎功能監測**：腎功能不全者需特別注意，可能造成高血鎂
4. **漸進增量**：從低劑量開始，觀察腸胃反應
5. **注意症狀**：高血鎂症狀包括低血壓、噁心、肌無力，應立即就醫

> ⚠️ **免責聲明**：本資訊僅供教育和研究目的，不構成醫療建議。任何用藥或補充劑變更應諮詢專業醫療人員。
