# 膠原蛋白選購指南

本指南協助您根據需求選擇適合的膠原蛋白保健食品。

## 為什麼選擇膠原蛋白？

### 與其他美容/關節保養方案比較

| 方案 | 優點 | 缺點 | 適合族群 |
|------|------|------|----------|
| **膠原蛋白** | 皮膚+關節+骨骼多效、研究充分、劑型多元 | 需持續補充、動物來源 | 抗老保養、關節保健 |
| 玻尿酸（口服） | 改善皮膚保水、關節潤滑 | 價格較高、分子大吸收有限 | 乾燥肌膚、乾燥型關節不適 |
| 一般蛋白質補充 | 價格低、胺基酸來源 | 非膠原蛋白專屬胺基酸 | 蛋白質攝取不足者 |
| UC-II®（非變性二型膠原蛋白） | 低劑量 40mg 有效、免疫調節機轉 | 價格高、僅針對關節 | 關節保養、運動族群 |
| 彈性蛋白（Elastin） | 改善皮膚彈性 | 研究較少、產品選擇少 | 進階皮膚保養 |

### 誰適合 / 誰不適合

**✅ 建議使用**：
- 25 歲以上開始抗老保養者
- 皮膚鬆弛、皺紋增加困擾者
- 關節不適、運動量大者
- 產後、更年期膠原蛋白流失加速者

**⚠️ 需諮詢醫師**：
- 腎功能不全者（需控制蛋白質攝取）
- 正在服用藥物者

**❌ 不建議使用**：
- 對牛、魚、雞過敏者（依來源選擇）
- 純素食者（可選擇膠原蛋白增強建造器）
- 期望立即見效者（需 8-12 週持續補充）

## 常見問題 FAQ

### Q1: 吃膠原蛋白真的有效嗎？
**A:** 是的，多項臨床研究顯示：
- 水解膠原蛋白可改善皮膚彈性（8-12 週見效）
- UC-II® 對關節不適有緩解效果
- 但效果因人而異，建議持續補充 2-3 個月評估

### Q2: 什麼時候吃最好？
**A:**
- **皮膚保養**：早上空腹或睡前
- **關節保養**：隨餐服用
- 可搭配維生素 C 一起服用提升吸收

### Q3: 牛膠原蛋白和魚膠原蛋白哪個好？
**A:**
- **牛膠原蛋白**：含第一、三型，研究最多，性價比高
- **魚膠原蛋白**：分子較小、吸收略佳，適合不吃牛者
- 效果差異不大，可依個人飲食習慣選擇

### Q4: 素食者怎麼辦？
**A:** 膠原蛋白來自動物，素食者可選擇：
- **膠原蛋白增強建造器**：含維生素 C、胺基酸、矽等合成原料
- 雖無法直接補充膠原蛋白，但可支持內源性合成

### Q5: 吃多少才夠？
**A:**
- 皮膚保養：每日 5-10g 水解膠原蛋白
- 關節保養：每日 40mg UC-II® 或 10g 水解膠原蛋白
- 軟糖、低劑量膠囊可能不足，建議選擇粉末

### Q6: 有副作用嗎？
**A:** 膠原蛋白安全性很高：
- 少數人可能有輕微腸胃不適
- 海洋來源者若對海鮮過敏應避免
- 不建議腎功能不全者大量攝取蛋白質
