# 乳酸桿菌

> 益生菌市場領導成分，腸道健康的守護者

## 什麼是乳酸桿菌？

乳酸桿菌（Lactobacillus）是一類革蘭氏陽性桿菌，屬於乳酸菌家族的核心成員。這類益生菌能將碳水化合物發酵產生乳酸，廣泛存在於人體腸道、口腔、陰道等部位，也常見於發酵食品如優格、泡菜、味噌中。

## 為什麼需要補充？

現代生活中的壓力、飲食不均、抗生素使用等因素可能破壞腸道菌群平衡，導致：
- 消化不適（脹氣、腹瀉、便秘）
- 免疫功能下降
- 營養吸收不良
- 情緒波動（腸腦軸線失衡）

補充益生菌有助於恢復腸道菌群平衡，維持整體健康。

## 常見菌株

| 菌株 | 特點 |
|------|------|
| L. acidophilus | 最經典的益生菌，耐胃酸能力佳 |
| L. rhamnosus GG | 研究最多，免疫調節證據豐富 |
| L. plantarum | 耐酸耐鹼，腸道定殖力強 |
| L. casei | 常見於優酪乳，幫助消化 |
| Bifidobacterium | 雙歧桿菌，常與乳酸桿菌搭配使用 |
