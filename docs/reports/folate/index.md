# 葉酸

> 維生素 B9，細胞分裂與 DNA 合成的關鍵營養素

## 什麼是葉酸？

葉酸（Folate）是維生素 B 群的一員，又稱維生素 B9。「Folate」泛指天然存在於食物中的各種形式，而「Folic Acid」則是人工合成的穩定形式，常見於營養補充劑和強化食品中。

葉酸在體內扮演關鍵角色：
- **DNA 合成與修復**：細胞分裂必需
- **紅血球生成**：預防巨球性貧血
- **神經管發育**：胎兒早期發育關鍵
- **同型半胱氨酸代謝**：心血管健康指標

## 為什麼需要補充？

### 高風險缺乏族群

| 族群 | 原因 |
|------|------|
| 備孕/懷孕女性 | 胎兒神經管發育需求大增 |
| 哺乳期女性 | 經乳汁流失 |
| 酗酒者 | 酒精干擾吸收與代謝 |
| MTHFR 基因變異者 | 無法有效轉化合成型葉酸 |
| 腸道疾病患者 | 吸收功能受損 |
| 長期服用特定藥物者 | 如 Methotrexate、抗癲癇藥 |

### 缺乏症狀

- 疲勞、虛弱
- 舌頭疼痛、口腔潰瘍
- 皮膚蒼白
- 呼吸急促
- 神經認知問題

## 市場概況

葉酸是全球保健食品市場的基礎成分，廣泛應用於：

| 產品類型 | 說明 |
|----------|------|
| 孕婦綜合維生素 | 標準配方必含成分 |
| B 群複方 | 與 B12 協同作用 |
| 心血管保健 | 搭配 B6、B12 調控同型半胱氨酸 |
| 活性葉酸 | 5-MTHF 形式，繞過 MTHFR |

## 葉酸形式比較

| 形式 | 說明 | 適合對象 |
|------|------|----------|
| Folic Acid | 合成形式，需體內轉化 | 一般人群 |
| 5-MTHF | 活性形式，直接利用 | MTHFR 變異者 |
| Folinic Acid | 中間代謝物 | 特定治療用途 |
