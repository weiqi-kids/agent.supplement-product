# 泛酸

> 維生素 B5，能量代謝與壓力調適的重要營養素

## 什麼是泛酸？

泛酸（Pantothenic Acid）是維生素 B 群的一員，又稱維生素 B5。「Pantothenic」源自希臘文「pantos」，意為「到處都有」，因為泛酸廣泛存在於各種食物中。泛酸是合成輔酶 A（Coenzyme A）的必要成分，參與體內多種代謝反應。

泛酸的主要功能：
- **能量代謝**：合成輔酶 A，參與脂肪酸代謝
- **荷爾蒙合成**：參與腎上腺皮質激素合成
- **神經傳導物質**：參與乙醯膽鹼合成
- **膽固醇合成**：參與膽固醇代謝途徑
- **皮膚健康**：維持皮膚屏障功能

## 為什麼需要補充？

### 可能需要補充的族群

| 族群 | 原因 |
|------|------|
| 高壓力族群 | 壓力增加泛酸消耗 |
| 痤瘡患者 | 高劑量 B5 可能有助皮脂調節 |
| 運動員 | 能量代謝需求增加 |
| 飲食不均衡者 | 攝取不足 |
| 酗酒者 | 吸收減少 |

### 缺乏症狀

泛酸缺乏罕見，因為廣泛存在於食物中，但嚴重缺乏可能導致：
- 疲勞、虛弱
- 頭痛
- 睡眠障礙
- 腸胃不適
- 手腳麻木或刺痛（灼足症候群）
- 情緒不穩

## 泛酸的形式

| 形式 | 說明 | 用途 |
|------|------|------|
| 泛酸鈣（Calcium Pantothenate） | 最常見的補充形式 | 口服補充 |
| 泛醇（Panthenol） | 前驅物，可轉化為泛酸 | 護膚、護髮 |
| 輔酶 A | 活性形式 | 體內代謝 |

## 市場概況

泛酸產品主要分為：

| 產品類型 | 說明 |
|----------|------|
| 單方 B5 | 針對特定需求（如痤瘡） |
| B 群複方 | 含標準劑量泛酸 |
| 抗壓配方 | 搭配其他抗壓成分 |
| 護膚/護髮產品 | 外用泛醇 |

## 劑量範圍

| 類型 | 劑量 |
|------|------|
| 每日建議量（成人） | 5 mg |
| 一般保健 | 5-10 mg |
| B 群複方常見劑量 | 10-100 mg |
| 痤瘡研究劑量 | 500-2000 mg |

> 泛酸為水溶性，目前無設定上限攝取量，但高劑量可能引起腹瀉
