# 納豆激酶選購指南

本指南協助您根據個人需求選擇適合的納豆激酶產品。

## 選購決策樹

```
我想改善血液循環
    │
    ├─ 一般保養
    │   └─ 納豆激酶 2000 FU/日
    │
    ├─ 加強循環
    │   └─ 納豆激酶 + 魚油 複方
    │
    └─ 同時調節血脂
        └─ 納豆激酶 + 紅麴 複方
            └─ 如台鹽納豆紅鞠
```

## 選購要點

### 1. 活性單位

| 活性 | 適用情況 | 說明 |
|------|----------|------|
| 2000 FU/日 | 一般保養 | 基礎劑量 |
| 4000 FU/日 | 加強保養 | 常見建議劑量 |
| > 4000 FU/日 | 特殊需求 | 應諮詢專業人員 |

**注意**：FU（Fibrinolytic Unit）是納豆激酶的活性單位，比重量（mg）更重要。

### 2. 純度與來源

| 條件 | 說明 |
|------|------|
| NSK-SD® | 知名專利原料，去除維生素 K |
| 非基改大豆 | 原料安全性 |
| 不含維生素 K2 | 避免影響抗凝血藥物 |

### 3. 複方搭配

| 搭配成分 | 功效 | 適合對象 |
|----------|------|----------|
| 紅麴 | 調節血脂 | 血脂偏高者 |
| 魚油 | 協同護心 | 心血管保養 |
| CoQ10 | 能量代謝 | 活力提升 |
| 銀杏 | 腦部循環 | 記憶力保養 |

### 4. 劑型選擇

| 劑型 | 特點 | 適合對象 |
|------|------|----------|
| 膠囊 | 最常見、保存性佳 | 一般人 |
| 腸溶膠囊 | 避免胃酸破壞 | 腸胃敏感者 |
| 軟膠囊 | 添加油脂、吸收佳 | 吸收不佳者 |

## 常見問題 FAQ

### Q1: 納豆激酶和吃納豆一樣嗎？
不完全一樣。納豆激酶是從納豆中萃取的酵素，濃度和活性都經過標準化。直接吃納豆也能攝取納豆激酶，但含量較不穩定。

### Q2: 納豆激酶要吃多久才有效？
通常 4-8 週可感受到循環改善。建議持續補充以維持效果。

### Q3: 納豆激酶可以和魚油一起吃嗎？
可以。兩者都有助於心血管健康，許多產品已做成複方。但若同時服用抗凝血藥物，應諮詢醫師。

### Q4: 台鹽納豆紅鞠的納豆激酶含量夠嗎？
台鹽納豆紅鞠結合納豆激酶與紅麴，適合想同時改善循環與血脂者。購買時可確認標示的 FU 活性單位。

### Q5: 吃納豆激酶會不會容易出血？
正常劑量下一般不會。但若有出血傾向、服用抗凝血藥物、或即將手術，應避免使用。

### Q6: 早上吃還是晚上吃比較好？
研究顯示晚上服用可能效果較佳，因為血栓較常在夜間形成。但空腹或隨餐服用皆可。

## 警語與注意事項

- 本指南僅供參考，不構成醫療建議
- **禁忌**：正在服用抗凝血藥物者（Warfarin、Aspirin 等）
- **禁忌**：手術前後 2 週
- **禁忌**：有出血性疾病者
- **禁忌**：孕婦、哺乳婦女
- 若出現異常出血、瘀青，應立即停用並就醫
