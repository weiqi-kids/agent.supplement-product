## 趨勢觀察

### 跨國共同趨勢

基礎維生素礦物質組合（Vitamin C + Calcium + Vitamin B6/B12 + Zinc + Magnesium）在美、加、韓三大市場持續穩居主流地位。本月 Zinc 上升至全球第二（43,462 筆），超越 Calcium（42,461 筆），顯示免疫健康訴求在三大市場的同步升溫，Zinc 在韓國更以 6,852 件穩居該國首位。Vitamin D 整合同義詞後達 40,889 筆位居第四，在美、加、韓三市場均位列前 5，反映了骨骼與免疫健康的持續關注。Probiotics 相關成分在韓國（2,051 件通稱登記）和北美持續強勢，腸道健康議題已成全球保健食品的核心訴求之一。

### 市場獨特趨勢

美國市場 DSLD 資料庫累計達 214,780 筆產品，其中 specialty（76,497 件）和 botanicals（67,449 件）兩大品類佔比最高，反映美國膳食補充劑市場的多元配方趨勢。韓國市場本月新增 86 筆 HFF 產品（44,584 升至 44,670），新增產品集中於 vitamins_minerals 及 specialty 品類，Zinc 以 6,852 件（較上月 4,898 件增長 40%）持續高居韓國首位，反映韓國消費者對免疫調節成分的偏好。日本市場 Indigestible Dextrin 合計達 486 件（FOSHU 350 件 + FNFC 136 件），GABA 成長至 132 件（上月 144 件為不同統計口徑，本月 FOSHU 新增 7 件為值得關注的信號，顯示 GABA 從 FNFC 擴展至 FOSHU 認證的趨勢）。加拿大市場以 Vitamin C（11,879 件）領先，Zinc（10,328 件）與 Vitamin D（10,206 件）緊追在後。

- **成分名稱標準化方法**：採用同義詞合併策略，將不同語言及學名俗名對應至統一標準名稱（如 Vitamin D3 = Cholecalciferol = コレカルシフェロール 統一計入 Vitamin D；Folic Acid 與 Folate 合併；Niacinamide 與 Niacin 合併；韓文 아연 對照至 Zinc）
- **日文成分名對照**：本月對照項目包含 GABA（γ-アミノ酪酸）、ルテイン（Lutein）、ヒアルロン酸Na（Hyaluronic Acid）、L-テアニン（L-Theanine）、難消化性デキストリン（Indigestible Dextrin）、葛の花由来イソフラボン（Kudzu Isoflavone）、茶カテキン（Tea Catechins）、モノグルコシルヘスペリジン（Monoglucosyl Hesperidin）等主要 FNFC/FOSHU 機能性成分
- **韓文成分名對照**：아연（Zinc）、비타민D（Vitamin D）、프로바이오틱스（Probiotics）、엽산（Folate）、철（Iron）、비타민B1（Vitamin B1）、나이아신（Niacin）、판토텐산（Pantothenic Acid）等韓文名稱對照英文標準名
- **已知限制**：
  - 美國 DSLD 包含完整的產品成分清單（含營養標示項目），本報告已過濾 Calories、Total Fat、Sodium、Total Carbohydrates 等純營養標示項目，僅保留具機能性的活性成分
  - 韓國 HFF 以 [] 標注機能性成分，統計邏輯與美加不同（美加為所有成分，韓國為申請認定的機能性成分），跨國絕對數量比較應考慮此差異
  - 加拿大 LNHPD 部分產品成分資訊以劑量格式列於 API（如 "Fish oil: 200.0 mg"），統計時以成分名稱歸併
  - 日本市場（FOSHU + FNFC 合計 2,603 件）資料量遠小於美加韓，跨國排名比較應考慮基數差異
