## 趨勢觀察

### 跨國共同趨勢

維生素類成分在美國、加拿大、韓國三大市場持續保持高需求，尤其是維生素 C（全球 1,862 件）、B 群和鋅（全球 1,622 件）。這反映消費者對基礎營養補充的穩定需求，特別是在後疫情時代對免疫支持的持續關注。

DHA/EPA 等 Omega-3 脂肪酸在美國、韓國、日本三大市場均有顯著登記數量（全球 DHA 707 件、EPA 298 件），顯示心血管健康和認知健康仍是全球保健食品的核心訴求。不同市場的宣稱重點略有差異：美國側重心臟健康，日本側重認知健康。

益生菌類產品在各市場均有穩定需求（Lactobacillus 全球 791 件、Bifidobacterium 全球 369 件），但菌株命名和登記方式存在顯著差異。韓國傾向以功能名稱登記，日本則以特定菌株名稱登記，這反映各國監管制度對益生菌的不同要求。

### 市場獨特趨勢

**日本市場**呈現最獨特的成分偏好。難消化性糊精（Indigestible Dextrin）以 490 件產品高居榜首，顯示日本消費者對腸道健康和血糖管理的高度關注。GABA（144 件）和銀杏葉（95 件）也是日本特色成分，分別對應放鬆/睡眠和認知健康訴求。異黃酮（87 件）則反映日本女性健康市場的特殊需求。本月新增的 FNFC 資料顯示，GABA、銀杏葉、葉黃素等成分主要來自 FNFC 體系。

**韓國市場**的益生菌產品占比顯著高於其他市場，且常以「프로바이오틱스」（Probiotics, 291 件）統一名稱登記而非具體菌株名。鋅（425 件）是韓國最熱門的維生素礦物質成分，可能與免疫支持需求相關。

**美國市場**以綜合配方和高劑量單方為特色，Proprietary Blend（專有配方, 678 件）的高登記量反映美國膳食補充劑市場的配方保護策略。硒（649 件）、鉻（643 件）、錳（589 件）等微量元素在美國市場份額較其他市場更高。

### 值得關注的成分

1. **GABA**（日本 144 件）— 日本 FNFC 中快速成長的成分，對應現代人壓力管理和睡眠品質需求。值得觀察是否會向其他亞洲市場擴散。

2. **Indigestible Dextrin（難消化性糊精）**（日本 490 件）— 日本獨大，但腸道健康趨勢全球化，此成分可能在其他市場獲得關注。

3. **Lutein（葉黃素）**（美國 236 件、日本 68 件）— 美國和日本均有顯著份額，隨著數位裝置使用增加，護眼成分可能持續成長。

4. **Bifidobacterium（比菲德氏菌）**（美國 307 件、日本 62 件）— 相較 Lactobacillus，Bifidobacterium 在日本市場份額更高，可能反映益生菌研究和消費者認知的地區差異。

5. **サーデンペプチド（沙丁魚肽）**（日本 54 件）— 日本獨有成分，主打血壓管理，代表日本市場對魚類來源機能性成分的偏好。

## 方法論說明

- **成分名稱標準化方法**：建立英文/日文/韓文同義詞對照表，優先轉換為英文通用名。同一成分的不同劑型或規格合併計算。
- **日文成分名對照**：難消化性デキストリン → Indigestible Dextrin、ビフィズス菌 → Bifidobacterium、ルテイン → Lutein、GABA/γ-アミノ酪酸 → GABA、イソフラボン → Isoflavone、茶カテキン → Tea Catechins、葉酸 → Folate、コラーゲン → Collagen、グルコサミン → Glucosamine 等。
- **韓文成分名對照**：비타민C → Vitamin C、비타민E → Vitamin E、프로바이오틱스 → Probiotics 等。
- **已知限制**：
  - 加拿大 LNHPD 成分資料需額外 API 擷取，本月僅 29 件產品有完整成分資料
  - 韓國健康機能食品成分資訊從「主要功能」欄位萃取，可能遺漏部分配方成分
  - 韓文維生素名稱（如 비타민B6, 비타민B1 等）尚未完全與英文名稱合併
  - 美國 DSLD 成分清單包含賦形劑，統計時已盡量過濾非活性成分
