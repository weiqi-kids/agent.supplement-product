# 關於系統

保健食品產品情報系統是一個跨國保健食品市場監測與成分趨勢分析平台。

## 系統目標

- 整合多國保健食品官方資料庫
- 自動化資料擷取與分析
- 提供定期市場報告與成分趨勢

## 資料處理流程

1. **擷取（Fetch）**：從各國官方資料庫下載原始資料
2. **萃取（Extract）**：解析原始資料，標準化產品資訊
3. **分析（Analyze）**：跨國比較、趨勢分析
4. **報告（Report）**：產出市場快照與成分雷達報告

## 更新頻率

- 資料擷取：每日
- 市場快照：每週
- 成分雷達：每月
